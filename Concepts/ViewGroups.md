# Viewgroup 
viewgroup is the container that holds multiple views.it helps us organise views on the screen 
for ex:
A layout that contains buttons and text fields
viewgroup manages and arrange views.
**Viewgroups in android**
* linearlayout(the arrangement of the viewgroup in horizontal or vertical.)
* relativelayout (positions view relative to each other or position the ui element as the parent is placed.)
  For ex:
  text 
  just below button 
* constraintlayout(we place multiple view and then attached them then place them into another view or parent view)
* framelayout(we literally are placing multiple view on each other.)
