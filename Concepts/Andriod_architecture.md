# Android Architecture
1.Application
2.Appframework
3.Android Runtime
4.Native Liberary
5.Linux Kernals
So the architecture is basically divided into 5 parts combining these 5 together whole application is created and works
1.Application:basically the first Ui user is interacting with.
2.Appframework:that is basically the Components and the service works together and give user desire output
3.Android Runtime:basically it's the core processor power for running apps
4.Native Liberaries:that are the libary used for the creation in media,graphic of the apps
5.linux kernel:the foundation and managing of the hardware like memory,battery etc.
