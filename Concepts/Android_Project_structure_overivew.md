# Android Project Structure Overview 
this basially have the 4 stages.
* Java Folder:this is the brain of the appplication basically have the source code(.java,.kt files)
* Resource folder:store the UI resource here(layouts,immage,strings,etc.)
* Manifest folder:Defines app permission components, metadeta, confriguration
* gradle scripts:it's handle Project libraries and build configuration(setting of the app and all.)
  this contain the build.gradle(project and module levels files)
